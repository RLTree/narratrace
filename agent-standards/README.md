# Agent Standards Modules

Load `../AGENT_STANDARDS.md` first, then load only the module that matches the
task. These files are operating law for narrated Record & Replay work.

- `01-namespace-and-progressive-disclosure.md`: file size, module boundaries,
  skill contract size, and path clarity.
- `02-boundaries-validation-and-enforcement.md`: evidence boundaries, parsing,
  validation, and mechanical gates.
- `03-execplans-worktrees-and-orchestration.md`: ExecPlans, worktree
  environment, and orchestration boundaries.
- `04-security-reliability-and-product-cohesion.md`: privacy, runtime proof,
  reliability, and product cohesion.
- `05-review-and-completion.md`: review team, claim ceiling,
  completion reports, plugin sync, and marketplace cache.
- `06-standards-gardening.md`: recurring friction, standards promotion, and
  cleanup cadence.
