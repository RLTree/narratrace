# Standards Gardening

## Standards Gardening

- Act immediately on recurring or severe misses: false completion, wrong review
  team/model/reasoning, unsafe orchestration, broken worktree initialization,
  product cohesion misses, trust-boundary gaps, oversized files, stale
  allowlists, and plugin-flow confusion that forced user micromanagement.
- Prefer the smallest durable fix: validator, schema, fixture, skill update,
  persona prompt update, standards module, plugin resource-map update, hook, or
  backlog row.
- Remove purposeless files instead of creating in-repo archives.
- If a recurring issue creates durable law, update `agent-standards/enforcement.json`,
  `agent-standards/enforcement.tsv`, and `agent-standards/enforcement-audit.tsv`
  in the same change.
