# Validation Artifacts

Use this directory for receipts, generated proof, and redacted run summaries.

Do not store raw audio, raw transcripts, secrets, broad logs, or unredacted Record & Replay artifacts here.

Current proof surfaces:

- `scripts/check` command output in chat closeouts.
- Future local receipt files for setup, runtime, redaction, and dogfood checks.
