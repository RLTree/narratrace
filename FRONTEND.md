# Frontend

There is no interactive browser, desktop, mobile, or visual control surface in
this skill directory yet. The skill can generate a static local
`review-artifact.html` for operator inspection, but that artifact is a rendered
review report, not a complete review UI or playback control surface.

## Future Trigger

Load this file before extending:

- a review UI;
- the static review artifact;
- a local dashboard;
- a playback or replay-time voice control surface;
- screenshots, accessibility proof, or product-cohesion receipts.

## Proof Spine Later

Future frontend work needs:

- local run command and URL or app surface;
- fixture data that does not contain raw private transcript content;
- keyboard and accessibility checks;
- screenshot or app-visible proof;
- product-cohesion receipt when claiming operator usefulness.

Until then, CLI, packet, and static review-artifact proof must not be described
as complete UI proof.
