# Generated Artifacts

Generated durable artifacts may be indexed here, but raw private run outputs do not belong here by default.

Allowed:

- redacted receipts;
- schema summaries;
- generated codemaps;
- distilled packet inspections.

Not allowed without explicit approval:

- raw audio;
- raw transcripts;
- unredacted Record & Replay logs;
- secrets or private workflow dumps.
