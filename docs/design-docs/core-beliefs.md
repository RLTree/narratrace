# Core Beliefs

- Evidence and narration are different materials.
- Time alignment is useful only when its confidence and clock assumptions are visible.
- Raw transcripts are private by default.
- The best packet is short enough for a future agent to use and strict enough to prevent false certainty.
- Product polish means reducing Tree's burden during capture and review, not adding decorative surfaces.
